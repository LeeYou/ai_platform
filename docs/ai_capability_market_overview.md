## 1. 产品定位

**AI 能力超市（AI Capability Market）** 是北京爱知之星科技股份有限公司面向企业级客户推出的 **一站式 AI 能力平台**，覆盖：

- 🔬 **AI 能力研发**：模型训练、评估、导出
- 🏪 **AI 能力交付**：标准化 REST API、SO/DLL 插件、Docker 镜像
- 🔒 **AI 能力授权**：机器绑定、能力粒度授权、有效期管理
- 🚀 **AI 能力运营**：多版本管理、热重载、全链路监控

平台提供 **105 种 AI 能力**，覆盖视觉、语音、NLP、文档智能四大领域，可根据客户需求灵活组合交付。

### 3.3 新增 AI 能力上线流程

```
  开发者
    │
    ├─① 在 train/scripts/<capability>/ 新增训练脚本
    │     train.py / export.py / config.json / requirements.txt
    │
    ├─② 在 cpp/capabilities/<capability>/ 实现 C++ 推理插件
    │     实现统一 C ABI: AiCreate / AiInit / AiInfer / AiDestroy ...
    │
    ├─③ 训练子系统 — 启动训练 → 导出模型包 (model.onnx + manifest.json)
    │
    ├─④ 测试子系统 — 批量精度评估，达到精度门槛
    │
    ├─⑤ 编译子系统 — 多平台编译 → libXXX.so / libXXX.dll 归档
    │
    ├─⑥ 授权子系统 — 在 License 配置中注册新能力，为客户签发授权
    │
    └─⑦ 生产系统 — 热重载，新能力立即对外提供 API 服务
```
